# portafolio
Se presentan una serie de soluciones a problemas de programación, para dar a conocer las capacidades técnicas en el lenguaje Java. En los códigos se comenta el enunciado a un problema, con ejemplos se salida que debe mostrar el programa y un link que redirige hacia el problema original de la página Codewars
Nota: Cuanto menor sea el Kyu, mayor la dificultad
